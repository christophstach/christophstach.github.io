Twitter datascience project

My datascience project is collection every tweet in Berlin. Tweets are saved to a MongoDB for further calculations. The calculations are made by another program, which is build with the Java SpringBoot framework, it is hosting a REST Web Service. It also utilises the MongoDB Aggregation framework and MongoDB mapReduce to perform very quickly. This website queries the API and shows the result of the calculations in charts generated by the Chart.js library. It is possible to narrow down the results to the date range selected in the input fields.
